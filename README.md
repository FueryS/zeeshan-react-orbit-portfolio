# Zeeshan React Orbit Portfolio

A modern multi-page developer portfolio built with React and Vite, designed around a futuristic React-inspired visual theme.

The portfolio showcases my skills, projects, achievements, and development journey through custom SVG animations, glassmorphism UI, orbit-based visual effects, and responsive layouts.

## Features

- Multi-page architecture using React Router
- Responsive design for mobile, tablet, and desktop
- Animated SVG Hero Orbit
- Interactive SVG Journey Timeline
- Skill Orbit visualization
- Glassmorphism UI components
- Animated gradient backgrounds
- Contact page with email integration
- Netlify and Vercel deployment support
